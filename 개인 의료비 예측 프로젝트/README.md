# 실시간 마케팅 반응고객 예측 및 감지조건 최적화
----

## 프로젝트 개요
```
· 트렌드가 빠르게 변화하는 유통업에서는 신규 고객을 확보하기 위해 계열사 이용 고객 대상으로 실시간 마케팅을 수행하고 있음.
· 실시간 마케팅을 효과적으로 활용하기 위해 데이터 분석 및 예측모델을 수행하려 함
· Ⅰ 구매가능성이 높은 고객군을 예측
· Ⅱ 최적의 감지조건의 규칙을 찾고 영업이익을 최대화하려 함 
```
## 분석 프로세스
```
· As-is : 각 지점의 마케터의 자율에 의해 고객군 선정과 감지조건 설정이 이루어짐
· To-be : ML 모델을 통해 구매 가능성이 높은 고객군을 추출하고 최적의 감지조건을 설정하여, 마케팅 반응률 향상을 기대
```

<h6>※ 1) AUROC : Area under ROC, 이진 분류기의 성능을 측정하기 위한 지표, 다양한 임계값에서 모델의 분류 성능을 측정함</h6>  
<h6>※ 2) LIFT : 향상도, 도출된 고객 군이 전체 구매율 대비 몇 배 더 구매 가능성이 높은지 측정하기 위한 지표</h6>

## 분석결과
```
· Probability을 기준으로 10분위수로 고객을 나누고, 마케팅 비용을 고려하여 몇 등급의 고객에게까지 마케팅을해야 최적의 ROI가 나오는지 도출
· 연관규칙분석을 통해 Domain Rule로 정의한 영업이익효율을 최대화 할 수 있는 감지조건 도출
```
